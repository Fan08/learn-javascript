### 1 正则表达式
是用于匹配字符串中字符组合的模式，正则表达式也是对象；
能被用于验证表单，过滤关键词，从字符串中提取特定部分；

### 正则表达式在js中的使用
#### 2.1 创建正则表达式
- 1 利用 RegExp 对象来创建正则表达式；
    ```
        const regexp = RegExp(/表达式/);
    ```
- 2 通过字面量创建；
    ```
        const regexp = /表达式/;
    ```

#### 2.2 测试正则表达式 test
test() 正则对象方法，用于检测字符串是否符合该规则，该对象会返回true或false，其参数是测试字符串；
```
    regexObj.test(str);
```
- 1 regexObj 是编写的正则表达式；
- 2 str 是要被测试的文本；
- 3 以上是测试 str 是否符合我们写的正则表达式；
